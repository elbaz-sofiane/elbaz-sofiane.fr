# 🌐 Portfolio – Sofiane Elbaz

Bienvenue sur le dépôt GitHub de mon site web **Portfolio** 🎯  

Ce projet a été réalisé afin de mettre en pratique mes compétences acquises en **E6 – Conception et Développement d’Application**, tout en construisant un espace professionnel pour présenter mes projets.

🔗 **Site en ligne :** [https://elbaz-sofiane.fr](https://elbaz-sofiane.fr)

---

## 🚀 Objectif du projet

Développer un site **Portfolio dynamique** permettant :

- 📂 La présentation de mes projets  
- 🌍 La mise à disposition de démonstrations en ligne  
- 💻 L’accès aux dépôts GitHub  
- 🔐 Une gestion complète via un espace administrateur sécurisé  

---

## 🛠️ Technologies utilisées

- **PHP**
- **MySQL**
- **HTML5**
- **CSS3**
- **JavaScript**
- Architecture **MVC simplifiée**
- Méthode **CRUD** (*Create, Read, Update, Delete*)

---

## 🔐 Espace Administrateur

Un panel administrateur sécurisé permet la gestion complète des projets :

| Action | Description |
|--------|------------|
| ➕ **CREATE** | Ajouter un nouveau projet |
| 👁️ **READ** | Consulter les projets existants |
| ✏️ **UPDATE** | Modifier un projet |
| 🗑️ **DELETE** | Supprimer un projet |

L'accès est protégé via un système d’authentification sécurisé.

---

## ✨ Fonctionnalités

- 💻 **Section Projets** avec :
